# Products API

Retrieves a product by its id.

Creates a new product.

<Route route="POST /1/products" />

## Update a Product

Updates a product.

<Route route="PATCH /1/products/:id" />

## Delete a Product

Deletes a product.

<Route route="DELETE /1/products/:id" />
