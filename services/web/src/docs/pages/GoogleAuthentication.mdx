# Google Authentication API

Validates Google tokens for authentication and user registration.

## Login

Verifies a Google token and authenticates the user.

<Route route="POST /1/auth/google/login" />

## Register

Verifies a Google token and creates a new user.

<Route route="POST /1/auth/google/register" />

## Enable

Verifies a Google token and registers a Google authenticator.

<Route route="POST /1/auth/google/enable" />

## Disable

Removes registered Google authenticator.

<Route route="POST /1/auth/google/disable" />
